# About Brutify:
Brutify it's a tool to bruteforce WPS/WPA/WPA2 Networks.
It's the first tool i publish, so be kind :D
# Installation:
1. Clone the repository:
*git clone https://github.com/ParzivalHack/Brutify*
2. Access the repository:
*cd Brutify*
3. Start the tool:
*python brutify.py*
4. Enjoy ;)
